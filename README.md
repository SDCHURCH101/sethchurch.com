# Seth Church | Alaska Entrepreneur & Civic Leader

A modern, responsive portfolio website showcasing Seth Church's professional accomplishments, enterprise leadership, and civic service in Alaska.

## About

This is a single-page static website (built with HTML, CSS, and JavaScript) featuring:
- **Professional Profile** — Seth Church's background as an Alaska entrepreneur and civic leader
- **Enterprise Overview** — Information about his 18-company enterprise spanning construction, real estate, and North Slope infrastructure
- **Civic Leadership** — Details on his service as a four-time gubernatorial appointee to the University of Alaska Board of Regents
- **Modern Design** — Cinematic hero section, bento-style stats, and responsive layout

## File Structure

- **`index.html`** — The main webpage (all content, styling, and interactivity in a single file for GitHub Pages)

## Accessing the Website

The website is published via GitHub Pages and is publicly accessible at:

**[https://SDCHURCH101.github.io/sethchurch.com/](https://SDCHURCH101.github.io/sethchurch.com/)**

## Deployment

This repository is configured for automatic deployment to GitHub Pages. Any changes pushed to the `main` branch will be reflected on the live website.
